Step by Step Guide to Building a PC

1. Install the CPU onto the Motherboard
2. Install M.2 SSDs (Optional)
3. Install Memory (RAM) on the Motherboard
4. Install CPU cooling
5. Get out case and all cables
6. Mount the power supply to the case
7. Install the motherboard
8. Install fans/water cooling radiator (Optional)
9. Attach all cables to motherboard, case, and power supply
10. Install the Graphics Card
11. Install any extra storage devices like HDDs, or SDDs
12. Install operating system with flash drive
